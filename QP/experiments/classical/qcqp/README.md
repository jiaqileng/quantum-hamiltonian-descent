# Environment Setup

To run 'suggest.py' you will need:
```bash
	python == 3.6
	numpy == 1.19.5
	scipy == 1.5.4
	CVXcanon >= 0.1.1
	cvxpy == 1.1.20
	scs == 2.0.2
```

To run 'improve.py' you will need:
```bash
	python == 3.6
	numpy == 1.13.3
	scipy == 0.19.1
	setuptools <=58.0.0
	cvxpy <= 0.4.9
	CVXcanon <= 0.1.1 (0.1.2 does not work!)
	qcqp == 0.8.3
```
